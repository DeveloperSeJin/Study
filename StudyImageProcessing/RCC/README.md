# Recommendations for Clothing Color Combinations

## 프로젝트 요약
> 사용자가 선택한 의상에 어울리는 색 조합을 편리하게 찾아주는 어플리케이션

## 기능 설명
1. 사용자가 불러온 이미지에서 매치하고 싶은 의상 클릭 시 의상의 색상과 어울리는 색상 추천
2. 추천한 색상 클릭 시 사용자가 직접 다른 의상의 색상 변경 가능
